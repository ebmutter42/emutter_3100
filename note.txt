Additional updates for this assignment.  I have cleared the database of all users and sessions so everything is fresh for your use.  Also, the /sessions.php endpoint has been updated to include three additional functions:
 
When passed a GET request with the parameter sessionID, the webservice will return a JSON object with the session details.
When passed a PUT request with the parameter sessionID, the webservice will return a JSON object with an Outcome or Error key.  This method updates the LastActivity field in the database
When passed a DELETE request with the parameter sessionID, the webservice will return a JSON object with an Outcome or Error key.  This method deletes the passed session\
